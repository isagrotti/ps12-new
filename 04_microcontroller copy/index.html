<div class="textcontainer"> <p class="margin"> </p> <h3>Week 4: Microcontroller Programming</h3> <h4>Programming</h4> <p class = "margin"></p> I used this as a opportunity to implement a dc motor to my kinetic sculpture. <pre><code class="language-arduino"> const int s = D0; // speed const int d = D1; // direction const int buttonPin = D2; // button pin const int freq = 5000; const int resolution = 8; void setup() { pinMode(s, OUTPUT); // outputs pinMode(d, OUTPUT); pinMode(buttonPin, INPUT); ledcAttach(s, freq, resolution); ledcWrite(d, 0); // motors off } void loop() { int buttonState = digitalRead(buttonPin); if (buttonState == HIGH) { motorA(HIGH, 63); // run when button is pressed } else { motorA(HIGH, 0); // stop if button not pressed } } void motorA(byte d, int s) { if (d == 1) { ledcWrite(s, 255 - s); digitalWrite(d, HIGH); } else if (d == 0) { ledcWrite(s, s); digitalWrite(d, LOW); } } </code></pre> <h5>Circuit schematic diagram:</h5> <img src="./schematic diagram.png" alt="schematic diagram" width="700"> </div>
